# """ Practical Questions Questions """

    Must function, low latency metrics/high quality

        How much data can you store?
        How much does it cost per token?
        How much does it cost to acquire a user?
        How many users can use the application simultaneously?
        How many total users can you support?
        How will users know about this application?
        What do users want from this application?
        What pain point does this application solve?

    Users need to be able to:
        Sign up
        Sign in
        Select avatars
        Log out
        Delete user data
        Delete all avatar data

    Additional considerations:
        How do you moderate content so it's not sued or criminalized?
        How do you share avatars?
        Users need API keys
        Users need rate limiting

    Potential usage and pricing:
        How many total users would potentially use this?
        How much would a user pay to use this?
        What is the user experience flow?
        How many users are available?
        How many users are immediately obtainable?
