# frontend with qr code for new users (you see a person, i present my phone, they scan the qr code, then they are brought through the following sign up process): 
new user scans the code and is brought to the frontend site with an avatar with live voice that will walk a new user through an onboarding process, instruct the user on the terms and conditions and privacy policy, gather a reference image and reference audio of the user, present options for avatars that are already created, integrations for their own avatar to be used, and ask if they would like to create an avatar of any other person by uploading images or other media from their phone. The user may select avatars from the gallery view and geo-located avatars for local businesses have discounts for use and will notify on proximity. The user is signed up for a free-tier, and is informed of follow up usage procedure to continue or alter their subscription via email once acquiesed.
