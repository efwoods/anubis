
# Each agent/avatar is a server-level architecture vs one server many assistants/avatars
each langgraph server is a docker image that is customized to the particular individual's real-world likeness.

each server in an authenticated server and each user in the neural nexus becomes authenticated to each server to communicate with the avatar. 

Each langgraph servier is cold-started so as to minimize cost, but becomes available to a user upon avatar selection and authentication of the user.

Each user has a list of authenticated servers they have access to stored in a db.
Each user may choose to share their server with the public 
Their are proprietary servers that everyone has access to. 

There are no assistants only a customized Anubis agent.
